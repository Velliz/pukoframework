Puko Framework is MVC PHP Framework for quick and fast PHP Application Development.



**this is [Core] module only for Puko Framework**

## Feature

* Puko Templating Engine (PTE)
* Puko Document Command (PDC)
* Puko Data Wrapper (PDW)
* URL Re-Routing
* Fast and Light
